# File Sorter (for Windows)

Just run the program and it will sort your files on the desktop into folders
- **Images**              -> '.png', '.jpg', '.jpeg', '.jfif', '.gif', '.apng', '.webp'

- **Media/Audio**         -> '.mp4', '.mkv', '.flv', '.mov', '.mp3', '.wav'

- **Text/Doc**            -> '.doc', '.docx', '.rtf', '.txt'

- **Coding**              -> '.py', '.html', '.js', '.c', '.cpp'
